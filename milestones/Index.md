# 🗺️ Implementação do Blog Tech — Índice de Milestones

> **Objetivo:** Executar a implementação do blog estático (Next.js + SQLite → GitHub Pages) em etapas sequenciais, com compilação e testes mínimos após cada passo.

## Pré-requisitos Gerais
- Ambiente Node.js 20+ (`node -v` ≥ 20)
- Git configurado (`git config --global user.email/name`)
- GitHub Pages habilitado no repositório (Settings → Pages → Source: `gh-pages` branch)
