# 📚 Milestones — Blog Tech

Este diretório contém a documentação estruturada da implementação do blog estático, organizada por milestones sequenciais.

## Como usar
1. Comece pelo [`Index.md`](./Index.md) para entender a sequência completa.
2. Cada arquivo `.md` descreve um passo com ações detalhadas e critérios de verificação.
3. Consulte as notas técnicas em [`TechnicalNotes.md`](./TechnicalNotes.md).
4. Após cada milestone, execute o teste mínimo indicado e avance ao próximo.
