# 🗺️ Milestone 05 — UI Completa, Tema Dark/Light e Layout Responsivo

**Objetivo:** Refinar a interface para ser moderna, simples, visualmente agradável e com tema escuro configurado.

| Tarefa | Detalhe | Entregável |
|--------|--------|----------|
| 4.1 | Configurar tema dark por padrão (ou switch) + suporte light | `themeColor` ou `prefers-color-scheme` no Next.js App Router |
| 4.2 | Criar layouts de página (single post, listagem, home) | Componentes reutilizáveis com shadcn/ui |
| 4.3 | Ajustes visuais e microinterações | UX refinada (hover, transições suaves, tipografia) |

**Critério de aceitação:** UI visualmente agradável em ambos modos dark/claro; responsivo para mobile; navegação fluida.
