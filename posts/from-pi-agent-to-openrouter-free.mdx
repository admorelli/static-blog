After Pi agent restored my sanity, I got curious: what if I could use the **best models available** without paying? Enter **OpenRouter's free tier**.

## The Setup

**Before:** Pi agent + hosted model (full precision, paid)  
**After:** Pi agent + OpenRouter free models (rotating: Llama 3.1, Nemotron 3 Ultra, Qwen 2.5, etc. — all free)  
**Agent:** Same Pi agent (6 focused tools)

## The Free Tier Gotchas

### Rate Limiting

```
Error: 429 Too Many Requests
Retry-After: 45s
```

During peak hours (US daytime), free tier gets throttled. My workaround:
- Batch requests
- Cache responses locally
- Accept 30-60s waits during peaks

### Context Window Truncation

Free models often have reduced context (32K vs 128K). For large codebases, this matters. My workaround:
- Feed only relevant files
- Summarize context before feeding
- Accept occasional context loss

### Model Rotation Surprises

Router sometimes switches models mid-conversation. The new model doesn't always pick up context perfectly. My workaround:
- Explicit context summaries in prompts
- "Previous context: [summary]"
- Accept occasional context resets

### Availability Windows

Some models unavailable during maintenance. Free tier = last priority. Had a 2-hour outage when Nemotron went down for maintenance.

## The "Model Roulette" Strategy

Since I can't control which free model handles each request, I developed a strategy:

```
1. Start with Nemotron (best reasoning)
2. If rate limited → Llama 3.1 (reliable, always available)
3. If Llama refuses → Qwen 2.5 (less aligned)
4. If all fail → wait 60s, retry
```

This "model ladder" handles 95% of cases.
